# Changelog

Все значимые изменения проекта документируются в этом файле.  
Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/).

## [2.0.0] — 2025-02-09

### Добавлено
- **Раздел настроек** (Ctrl+, или Вид → Настройки):
  - Настройка глобальных горячих клавиш (Старт/Стоп, Запись, Экстренная остановка)
  - Общие настройки (поведение, внешний вид, логирование)
  - Страница «О программе»
- **Режим зажатия клавиши** (Hold Key) — удержание клавиши до остановки
- **SVG-иконки** для вкладок умных режимов (вместо нечитаемых emoji)
- **Английская документация** (docs/USER_GUIDE_EN.md)

### Исправлено
- Вкладки умных режимов теперь читаемы (заменены emoji на SVG-иконки)
- Переключение языка теперь работает корректно (EN↔RU)
- Удалены все emoji из интерфейса — кроссплатформенная совместимость
- Горячие клавиши загружаются из настроек (ранее были захардкожены F6/F7/F8)

### Изменено
- Очищен весь UI от emoji-символов для стабильного рендеринга
- Обновлены файлы переводов (app_en.ts, app_ru.ts) для соответствия новым строкам
- Горячие клавиши настраиваемы через интерфейс настроек

## [1.0.0] — 2025-02-08

### Добавлено

#### Модуль мыши
- Автоклик ЛКМ / ПКМ / СКМ с настраиваемым интервалом (от 1 мс)
- Режимы: одиночный клик, двойной клик, удержание
- Клик по фиксированным координатам или следование за курсором
- Мультиточечный клик — последовательность координат (scripted path)
- Рандомизация интервала и координат (анти-детект)
- Ограничение по количеству кликов или по времени
- Захват координат щелчком мыши

#### Модуль клавиатуры
- Автонажатие любой клавиши с настраиваемым интервалом
- Автонажатие комбинаций клавиш (Ctrl/Shift/Alt/Win + клавиша)
- Режим печати текста (строка → посимвольный ввод с задержкой)
- Макрос-режим: запись и воспроизведение последовательности нажатий
- Рандомизация задержки между нажатиями
- Захват клавиши в реальном времени

#### Макро-движок
- Визуальный редактор макросов с Drag & Drop
- 16 типов действий: мышь, клавиатура, задержки, циклы, вложенные макросы
- Запись макросов в реальном времени (мышь + клавиатура + задержки)
- Воспроизведение с настройкой скорости (0.1x – 10x) и количества повторов
- Пауза/возобновление воспроизведения
- Циклы с вложенностью и условным выполнением
- Экспорт/импорт макросов (JSON-формат)
- Библиотека макросов с управлением (создание, удаление, переименование)

#### Умные режимы
- Привязка к окну (работа только в определённом окне)
- Поиск и клик по цвету пикселя с настройкой допуска
- Поиск по изображению (template matching без OpenCV)
- Планировщик задач: запуск макросов по расписанию
- Профили: сохранение/загрузка всех настроек

#### Интерфейс и UX
- Современный нативный GUI на Qt6 (минимальное потребление ресурсов)
- Тёмная тема (Catppuccin Mocha) и светлая тема (Catppuccin Latte)
- System tray с управлением (старт/стоп, запись, экстренная остановка)
- Глобальные хоткеи: F6 (старт/стоп), F7 (запись), F8 (экстренная остановка)
- Панель логирования в реальном времени (с сохранением в файл)
- Мультиязычность: Русский и Английский

#### Системное
- Поддержка Windows 10/11 (64-бит)
- Статическая линковка CRT (портативная сборка без зависимостей)
- Портативный режим (настройки рядом с .exe)
- NSIS-установщик с автозапуском и ярлыками
- LTO-оптимизации для минимального размера
- Скрипты автоматической сборки (build.bat, package.bat)
- CI/CD через GitHub Actions
